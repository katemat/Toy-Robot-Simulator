Toy Robot Simulator

Description

- The application is a simulation of a toy robot moving on a square tabletop, of dimensions 5 units x 5 units.
- There are no other obstructions on the table surface.
- The robot is free to roam around the surface of the table, but must be prevented from falling to destruction.
- The first valid command to the robot is a PLACE command

Any movement that would result in the robot falling from the table must be prevented, however further valid movement commands must still be allowed.

Commands:

  • PLACE X,Y,F- X,Y coordinates between 0 and 4 included, F - facing: NORTH, EAST, SOUTH, WEST
  • MOVE - move the toy robot one unit forward in the direction it is currently facing
  • LEFT- rotate the robot 90 degrees in the left direction
  • RIGHT- rotate the robot 90 degrees in the right direction
  • REPORT- announce the X,Y and F of the robot
